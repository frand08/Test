Using a S\+T\+M32\+F103, this test reads magnetometer and temperature values from the L\+I\+S3\+M\+DL and stores every second in the F\+L\+A\+SH memory W25\+Q80\+DV, assigning a unique ID on each value. In case a message is received from the U\+A\+R\+T1 and it matches with any of the I\+Ds stored in the memory, the M\+CU will transmit via U\+A\+R\+T1 the data, otherwise it will give an error.
